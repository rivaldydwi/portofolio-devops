How this pipeline grew

PHASE 01
AWS
Started with AWS EC2
Built the first working CI/CD pipeline. GitHub Actions triggered on every push, built a Docker image, and deployed it to an EC2 instance. It worked — but the monthly bill didn't.
GitHub Actions Docker AWS EC2
PHASE 02
SELF-HOSTED
Moved to a Mini PC
Cut cloud costs entirely by deploying to a self-hosted server at home. Used Tailscale VPN to connect the GitHub Actions runner securely — no public port exposure needed.
Tailscale VPN Self-hosted SSH Deploy
PHASE 03
NOW
Added IaC with Terraform
Replaced manual docker run commands with Terraform. Infrastructure is now version-controlled, consistent on every deploy, and smart enough to skip restarts when nothing has changed. Also added Trivy scanning at two checkpoints.
Terraform Trivy IaC Docker Hub Slack Notify
